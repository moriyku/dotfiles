# dotfiles

## Setup

### Linux

```
cd ~
git clone https://github.com/moriyku/dotfiles.git

chmod +x dotfiles/setup.sh
dotfiles/setup.sh
```

### Windows

- **Open PowerShell as Administrator**

```
cd ~
git clone https://github.com/moriyku/dotfiles.git

dotfiles\setup.ps1
```

## Checking Differences with Existing Dotfiles

### Linux

```
chmod +x dotfiles/diff.sh
dotfiles/diff.sh
```

### Windows

```
dotfiles\diff.ps1
```
